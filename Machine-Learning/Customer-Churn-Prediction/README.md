# Customer Churn Prediction

## Overview

This project focuses on developing a Machine Learning model to predict customer churn.

The objective is to analyze customer behavior, identify relevant patterns, and build a classification model capable of predicting customers with a higher probability of leaving a service.

## Tools & Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Jupyter Notebook

## Project Workflow

1. Data exploration
2. Data cleaning and preprocessing
3. Feature analysis
4. Model training
5. Model evaluation
6. Business interpretation of results

## Machine Learning Models

- Decision Tree Classifier
- Model evaluation using accuracy and other metrics

## Results

The model identifies relevant patterns associated with customer churn and demonstrates the application of machine learning techniques for business decision-making.
